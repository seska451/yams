
Use the `zone` module to find, create and manage zone.

***

### zone:new

!!! example
    ```lua
    yams.zone:()
    ```

***

### zone:validate

!!! example
    ```lua
    yams.zone:validate()

    ```

***

### zone:set_name

!!! example
    ```lua
    yams.zone:set_name(name)

    ```

***

### zone:set_zone_type

!!! example
    ```lua
    yams.zone:set_zone_type(zone_type)

    ```

***

### zone:find

!!! example
    ```lua
    yams.zone:find(name)

    ```
